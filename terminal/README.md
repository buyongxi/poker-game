# 德州扑克终端前端

基于 Python Rich 库的命令行德州扑克游戏界面，支持在线联网和离线本地游戏。

## 快速开始

### 安装

```bash
cd terminal
pip install -r requirements.txt
```

### 运行

#### 完整模式（需要后端服务）

```bash
python main.py
```

这将启动完整的应用程序，包括：
1. 登录/注册界面
2. 游戏大厅
3. 德州扑克游戏

#### 离线模式（无需后端）

```bash
python poker_terminal.py
```

这将直接启动本地游戏（带简单机器人）。

## 游戏说明

### 操作流程

1. **启动游戏** - 运行 `python main.py`
2. **登录/注册** - 选择登录方式或注册新账号
3. **游戏大厅** - 创建或加入房间
4. **进行游戏** - 轮到你时选择操作

### 玩家操作

| 操作 | 说明 | 快捷键 |
|------|------|--------|
| 弃牌 (Fold) | 放弃当前手牌 | [1] |
| 过牌 (Check) | 不下注通过 | [2] |
| 跟注 (Call) | 跟随当前下注 | [3] |
| 加注 (Raise) | 增加下注金额 | [4] |
| 全押 (All-in) | 押上所有筹码 | [5] |

### 牌型大小

```
皇家同花顺 > 同花顺 > 四条 > 葫芦 > 同花 > 顺子 > 三条 > 两对 > 一对 > 高牌
```

## 后端 API

完整模式需要后端服务运行在 `http://localhost:8000`。

### API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| /api/auth/register | POST | 用户注册 |
| /api/auth/login | POST | 用户登录 |
| /api/auth/me | GET | 获取当前用户 |
| /api/rooms | GET | 获取房间列表 |
| /api/rooms | POST | 创建房间 |
| /api/rooms/{id}/join | POST | 加入房间 |

## 注意事项

1. 离线模式下所有数据在内存中运行，退出后不会保存
2. 建议终端窗口宽度至少 80 字符
3. 支持真彩色终端可获得最佳显示效果
